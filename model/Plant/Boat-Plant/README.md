# Boat Plant Model
This model is contributed by shaolong Yang、yanhao Sun、xingyun Gao、shuo Sun and referenced from A rapid development method of autonomous navigation controller for unmanned ship based on ROS hardware-in-the-loop simulation. It is an underactuated unmanned surface vessel (USV) model featuring a single thruster and a rudder, supporting only forward motion and heading motion.

## Environment
- MATLAB 2018b or higher
